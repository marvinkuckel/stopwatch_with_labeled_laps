# stopwatch_with_labeled_laps
Mostly generated with Claude Sonnet 4.5

A Python Kivy application implementing a stopwatch with lap functionality,
extended with the ability to create custom labels and assigning them to a lap.

All you need is `pip install kivy`, preferably kivy==1.9.1

For Android build: buildozer<br>
For iOS build: pyobjus
